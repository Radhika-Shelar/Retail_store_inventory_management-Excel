# Retail_store_inventory_management-Excel



## Project Overview

This project focuses on analyzing retail inventory data using Microsoft Excel to help businesses monitor inventory levels, reduce stock shortages, and improve inventory management decisions.

The dashboard provides interactive insights into stock value, supplier performance, product categories, and restocking schedules through Pivot Tables, Pivot Charts, KPIs, and Slicers.

---

## Problem Statement

Retail stores often face challenges such as:

- Overstocking, which increases storage costs.
- Stock shortages, leading to lost sales.
- Manual inventory tracking, which is time-consuming and error-prone.

This project aims to build an interactive inventory dashboard for better monitoring and data-driven decision making.

---

## Objectives

- Monitor inventory levels
- Identify low-stock products
- Track upcoming restocks
- Analyze inventory by category and supplier
- Create an interactive dashboard for business insights

## Project Workflow

1. Imported the dataset into Excel.
2. Removed duplicate records.
3. Cleaned and formatted the data.
4. Created calculated columns:
   - Total Inventory Value
   - Stock Status
5. Applied Conditional Formatting.
6. Created Pivot Tables.
7. Built Pivot Charts.
8. Designed KPI Cards.
9. Added interactive Slicers.
10. Answered business questions using dashboard insights.

## Key Insights

- Toys category has the highest inventory value.
- Omni Suppliers contribute the highest-value inventory.
- Online Store holds the highest inventory value.
- Around 30% of products are classified as Low Stock.
- No products are overdue for restocking.
- Average Unit Cost is approximately ₹9,967.85.
